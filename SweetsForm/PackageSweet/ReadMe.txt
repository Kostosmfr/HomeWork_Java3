Если в корневой папке нет файла SweetsPackage, то открывается форма Choice Sweets, иначе открывается форма Create File.
В форме Create File:
 - Можно создать новый "пакет" (кнопка "Собрать новый") - открывается форма Choice Sweets, существующий пакет переименовывается в SweetsPackage + yyyyMMdd_hhmmss и перекладывается в папку BackUp

 - Добавить в старый "пакет" (кнопка "Добавить в старый") - открывается форма Choice Sweets
 - Закрыть - закрываем программу

В форме Choice Sweets:
 1) Выбираем из ChoiceBox тип продукта
 2) Становятся доступные поля для ввода характеристик
 3) Добавляем продукт в пакет:
	a) В элементе TextArea отображаются главные характеристики
	b) Под элементом TextArea в Label складывается общая цена и общий вес
	c) Происходит создание, если файл SweetsPackage отсутствует, файла и запись продукта в файл. Если файл отсутствует, происходит добавление продукта в "пакет".

Фичи:
Подсказка при наведении курсора мыши на элемент ChoiceBox
